Phabrix Automation Library Manual
=================================

:Date: |today|

This guide describes how the Phabrix automation library can be used with a test framework such as pytest
to build functional regression tests that control a variety of devices. It also demonstrates how
the automation library can be used for the creation of useful command line tools, test GUIs etc.
